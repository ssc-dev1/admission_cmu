Odoo 14 Web Tinymce Editor (Community)

Web tinymce editor module allow user to access odoo html field functionality as well as extra more features.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Available for Enterprise

How to use

  • General Settings -> Enable Tinymce Editor.

  • As per below image, we can see new editor which is define using simple html field or also we can define as a widget 'html'.
